    Xlsxir v1.6.3 API Reference

    Modules

    Xlsxir

    Extracts and parses data from a .xlsx file to an Erlang Term Storage (ETS) process and provides various functions for accessing the data

    Xlsxir.ConvertDate

    Converts an ISO 8601 date format serial number, in char_list format, to a date formatted in Erlang :calendar.date() type format (i.e. {year, month, day})

    Xlsxir.ConvertDateTime

    Converts a datetime formatted as a decimal number that represents the fraction of the day in char_list form, to an elixir naive datetime

    Xlsxir.ParseString

    Holds the SAX event instructions for parsing sharedString data via Xlsxir.SaxParser.parse/2

    Xlsxir.ParseStyle

    Holds the SAX event instructions for parsing style data via Xlsxir.SaxParser.parse/2

    Xlsxir.ParseWorkbook

    Holds the SAX event instructions for parsing style data via Xlsxir.SaxParser.parse/2

    Xlsxir.ParseWorksheet

    Holds the SAX event instructions for parsing worksheet data via Xlsxir.SaxParser.parse/2

    Xlsxir.SaxParser

    Provides SAX (Simple API for XML) parsing functionality of the .xlsx file via the Erlsom Erlang library. SAX (Simple API for XML) is an event-driven parsing algorithm for parsing large XML files in chunks, preventing the need to load the entire DOM into memory. Current chunk size is set to 10,000

    Xlsxir.StateManager

    GenServer to manage ETS table state/

    Xlsxir.StreamWorksheet

    Holds the special SAX event instructions for parsing and streaming worksheet data rows via Xlsxir.SaxParser.parse/2

    Xlsxir.Unzip

    Provides validation of accepted file types for file path, extracts required .xlsx contents to memory or files

    Xlsxir.XlsxFile

    Struct and helper functions to extract and process .xslx files

    Xlsxir.XmlFile

    Struct that represents an XML file extracted from an xlsx file, either in memory (in the content field) or on the filesystem (located in the path field)
